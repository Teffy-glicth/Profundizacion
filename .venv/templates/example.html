Casos de Uso del Machine learning supervisado

¿Qué es?

el machine learning supervisado es un tipo de machine Learning en el que se entrena a un algoritmo con grandes cantidades de datos etiquetados. El machine learning supervisado le enseña al algoritmo a predecir resultados con precisión. Este tipo de machine learning es el más utilizado.

¿Como funciona?

Funciona a través de reconocimiento de patrones mediante el análisis de datos etiquetados, el algoritmo detecta patrones y genera predicciones que se basan en patrones derivados de cuando se le consulta.

La calidad de los resultados esta directamente relacionada con la calidad de los datos con los que se haya entrenado

Casos de uso

Los casos de uso del Machine Learning supervisado son:

  1. Marketing y ventas: Se utilizan para la generacion de clientes potenciales, análisis de datos, busquedas en linea y la optimizaciób de motores de búsqueda. Algunos ejermplos podrian ser plataformas de streaming como Netflix, Amazon estre muchas otras.
  2. Servicio al cliente: Existen dos formas en las que el ML puede ser utilizado en el servicio al cliente. El primero las consultas por voz, en el que el ML es capaz de entender al cliente y comprender su tono de voz de esta manera los dirige a los asesores adecuados. La segunda forma se basa en las consultas por textos atendidas por chatbots, los cuales sirven para optene una atención inmediata las 24 horas del día.
  3. Asistentes personales y de voz: Los ML impulsan las tareas de los asistentes personales o de voz como Alexa o Siri. Esto implica el reconocimiento de voz la conversion de voz a texto o de texto a voz. en el caso de que alguien llegue a realizar una pregunta el ML busca la respuesta o es capaz de recordar preguntas similares que se hayan hecho antes
